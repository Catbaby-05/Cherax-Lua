# Recovery+ V1.1

## Description

Adds extra recovery options to the menu.

## Options

Recovery > unlocks:
- Fast run
- Max mental state
- Reset mental state

Recovery > rank:
- Change player deaths (k/d ratio)
- Change player kills (k/d ratio)
- Change race losses
- Change race wins
- Change total playtime (in hours)
- Change deathmatch losses
- Change deathmatch wins
